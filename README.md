# projeto-api-public
Desenvolvimento projeto educacional da disciplina de projetos. Foco: assimilar a gestão por meio de ferramentas de controle.

Roteiro:
1. Kick-Off (Reunião de Ler)
2. Criação de TAP
2.1 Validação da TAP
2.2 Criar o Projects
3. Criação das Issues (tasks)
   3.1 Atrelar responsáveis (Quem responde pela Task)
   3.1.2 Criar os milestones (depende de cada projeto)
3.2 Atrelar ao Projects
4. Distribuir ao Diagrama de Gantt
5. Iniciar a execução
